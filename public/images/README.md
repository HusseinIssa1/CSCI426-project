## How to Add Images

1. Copy all image files from the root `images` folder to this `public/images` folder
2. Ensure image file names match exactly as listed above
3. Images will be automatically served by React from this location

## Note

In React, images in the `public` folder are served from the root URL, so:
- Path in code: `images/filename.jpg`
- Actual location: `public/images/filename.jpg`
- URL in browser: `/images/filename.jpg`
